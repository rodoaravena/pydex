# PyDex Lite

PokéDex para la consola de Python

## Consideraciones antes de ejecutar

* Tener instalado **Python 3.5 o superior**

## Ejecutar el proyecto en Windows

### Usando Python IDLE

1. Ir al Menú Inicio
2. Abrir IDLE
3. Hacer clic en *File* y luego en *Open* y abrir el archivo `pydex.py` que está en la carpeta del proyecto
4. Luego ir a *Run* y hacer clic en *Run module F5*
5. Se ejecutará el proyecto

### Usando PowerShell

1. Ir a la carpeta del proyecto
2. Hacer clic en *Archivo* y luego en *Abrir Windows PowerShell*
3. Escribir el comando `py pydex.py`
4. Se ejecutará el proyecto

## Ejecutar el proyecto en Linux o macOS

### Usando una Terminal

1. Abrir una terminal en la carpeta del proyecto
2. Escribir `python3 pydex.py`
3. Se ejecutará el proyecto

## Ejemplo de salida

**Output:**

```bash

Bienvenido a PyDex
Ingrese el nombre del Pokémon:
```

